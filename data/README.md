# Dataset

The dataset used in this project is the **Mobile Price Prediction** dataset.

**Source:** https://www.kaggle.com/datasets/fhabibimoghaddam/mobile-price-prediction

**Author:** Habibimoghaddam, F. (2023)

## How to Download

1. Visit the Kaggle link above
2. Sign in to your Kaggle account
3. Click "Download" to get the CSV file
4. Place the downloaded file in this `data/` folder
5. Rename it to `mobile_price.csv` if needed

## Dataset Details

- **Samples:** 2,000 mobile phones
- **Features:** 20 technical attributes
- **Target Variable:** price_range (0=Low, 1=Medium, 2=High, 3=Very High)
- **Class Distribution:** Perfectly balanced (25% each class)

## Features Include

| Feature | Description |
|---------|-------------|
| battery_power | Battery capacity (mAh) |
| ram | Random Access Memory (MB) |
| px_height | Pixel height |
| px_width | Pixel width |
| fc | Front camera (MP) |
| pc | Primary camera (MP) |
| clock_speed | Processor speed (GHz) |
| n_cores | Number of processor cores |
| int_memory | Internal memory (GB) |
| mobile_wt | Mobile weight |
| sc_h | Screen height (cm) |
| sc_w | Screen width (cm) |
| talk_time | Talk time (hours) |
| four_g | 4G support (binary) |
| three_g | 3G support (binary) |
| wifi | WiFi support (binary) |
| bluetooth | Bluetooth support (binary) |
| dual_sim | Dual SIM support (binary) |
| touch_screen | Touch screen (binary) |
| m_dep | Mobile depth (cm) |
